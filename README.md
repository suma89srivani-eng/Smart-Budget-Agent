# 💰 Smart Budget Agent

Smart Budget Agent is a smart AI-based budgeting assistant that helps users track expenses, manage monthly budgets, analyze spending patterns, and receive financial insights through an interactive interface.

## 🚀 Features

- 💸 Expense tracking
- 📊 Budget monitoring
- 🧠 Spending insights
- 📅 Monthly budget planning
- 🎨 Clean and user-friendly UI
